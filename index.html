Bring Your Own Key

Promptron uses your own Anthropic API key. Model requests are sent directly from your device to Anthropic. Promptron does not proxy or resell model access.

Local Workspace

Projects, conversations, generated prompts, scores, and local account data are stored on your device. Promptron does not run analytics, ads, tracking, or third-party SDKs.

Promptron Pro

Pro unlocks app features: all product-line forms, full encyclopedia retrieval, multi-project workspaces, live evaluation, and export. Anthropic usage is billed separately to your own API key.

Common Questions

Do I need an Anthropic API key?

Yes for live generation. Without an Anthropic key, Promptron can still be inspected, but live model calls and live evaluation require your own Anthropic account and API key.

Does Promptron collect my prompts?

No. Promptron is designed around on-device storage and BYOK direct requests to Anthropic. Prompt content is not sent to Promptron servers.

Can I cancel Pro?

Yes. Subscriptions purchased through Apple are managed in your Apple account: System Settings or App Store account settings → Subscriptions.

Does Promptron promise better prompts?

No. Promptron is positioned honestly: it sells convenience, product-line fit, local organization, retrieval, and evaluation workflow, not guaranteed prompt superiority.
